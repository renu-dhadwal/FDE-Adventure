3.1 Rigorous Definitions

Acharya Anvaya: Let us be precise. Suppose \(n-1 < \alpha \leq n\) with \(n \in \mathbb{N}\).

Riemannâ€“Liouville fractional derivative

\[ D_t^\alpha f(t) = \frac{d^n}{dt^n}\, I^{\,n-\alpha} f(t) = \frac{1}{\Gamma(n-\alpha)}\frac{d^n}{dt^n}\int_0^t (t-\tau)^{\,n-\alpha-1} f(\tau)\,d\tau. \]

Caputo fractional derivative

\[ {}^C D_t^\alpha f(t) = I^{\,n-\alpha} f^{(n)}(t) = \frac{1}{\Gamma(n-\alpha)}\int_0^t (t-\tau)^{\,n-\alpha-1} f^{(n)}(\tau)\,d\tau. \]

Ananya: So the difference is: Riemannâ€“Liouville differentiates after integrating, while Caputo differentiates before integrating?

Acharya Anvaya: Exactly. Two boats across the same river â€” same waters, different journeys ðŸ˜‡.

3.3 Worked Example: Power Functions

For the Riemannâ€“Liouville derivative of \(t^\beta\) (\(\beta > -1\)):

\[ D_t^\alpha \, t^\beta = \frac{\Gamma(\beta+1)}{\Gamma(\beta-\alpha+1)} \, t^{\beta-\alpha}. \]

Example: with \(\alpha = \tfrac{1}{2}\) and \(f(t)=t^2\):

\[ D_t^{1/2}\, t^2 = \frac{\Gamma(3)}{\Gamma(2.5)} \, t^{3/2}. \]

3.4 Worked Example: Caputo Derivative of Power Functions

  • If \(f(t) = t^\beta\) with \(\beta \leq n-1\), then \({}^C D_t^\alpha f(t) = 0\).
  • If \(\beta > n-1\), then

\[ {}^C D_t^\alpha t^\beta = \frac{\Gamma(\beta+1)}{\Gamma(\beta-\alpha+1)}\, t^{\beta-\alpha}. \]

So for most powers, Riemannâ€“Liouville and Caputo agree, but for low-degree polynomials, Caputo derivatives vanish.

3.6 Rules of the River (Caputo Properties)

Rule 1. Linearity

For constants \(c_1, c_2\) and functions \(f, g\):

\[ {}^C D_t^\alpha (c_1 f(t) + c_2 g(t)) = c_1 \, {}^C D_t^\alpha f(t) + c_2 \, {}^C D_t^\alpha g(t). \]

Rule 2. Constants and Polynomials

  • For a constant \(C\), \({}^C D_t^\alpha C = 0\).
  • For \(t^\beta\) with \(\beta = 0,1,\dots,n-1\), \({}^C D_t^\alpha t^\beta = 0\).
  • For \(\beta > n-1\),
    \[ {}^C D_t^\alpha t^\beta = \frac{\Gamma(\beta+1)}{\Gamma(\beta-\alpha+1)}\, t^{\beta-\alpha}. \]

Rule 3. Exponentials and Trigonometric Functions

  • \({}^C D_t^\alpha e^{\lambda t} = \lambda^\alpha e^{\lambda t}\).
  • \({}^C D_t^\alpha \sin(\omega t) = \omega^\alpha \sin\!\left(\omega t + \tfrac{\pi\alpha}{2}\right)\).
  • \({}^C D_t^\alpha \cos(\omega t) = \omega^\alpha \cos\!\left(\omega t + \tfrac{\pi\alpha}{2}\right)\).

Rule 4. Product and Chain Rules (Caution!)

Unlike classical derivatives, Caputo derivatives do not have simple product or chain rules. Use linearity safely, but treat products/compositions with care.
